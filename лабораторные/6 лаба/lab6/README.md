# Лабораторная работа №6 - Технология AJAX

## Описание проекта

Данная лабораторная работа демонстрирует использование технологии AJAX для асинхронного обмена данными с сервером без перезагрузки страницы.

## Структура проекта

```
project/
│
├── ajax-search.html    # Задание 1: Поиск с ленивой загрузкой
├── news.html           # Задание 2: Новостная страница
├── ajax.php            # Серверный скрипт для обработки AJAX-запросов
└── README.md           # Документация проекта
```

## Установка и настройка

### 1. Установка веб-сервера

- Установите XAMPP.

### 2. Проверка работы сервера

1. Запустите XAMPP Control Panel
2. Запустите Apache
3. Открыть ajax-search.html и news.html в браузере
4. Вы должны увидеть JSON-данные с новостями

## Описание заданий

### Задание 1: AJAX Поиск (ajax-search.html)

**Функционал:**
- Поиск данных через AJAX запросы
- Отображение результатов по 10 элементов
- Ленивая загрузка (lazy loading) - кнопка "Загрузить ещё"
- Работа с уже загруженными данными без повторных запросов

**Использование:**
1. Откройте `http://localhost/ajax-lab/ajax-search.html`
2. Введите запрос в поле поиска
3. Нажмите кнопку поиска или Enter
4. При наличии более 10 результатов появится кнопка "Загрузить ещё"

**Технические особенности:**
- POST запрос к `ajax.php?type=nickname`
- Кэширование всех загруженных данных
- Постепенное отображение по 10 элементов
- Счетчик оставшихся результатов

### Задание 2: Новостная страница (news.html)

**Функционал:**
- Автоматическая загрузка новостей при открытии страницы
- Постраничная навигация (5 новостей на странице)
- Переключение страниц без перезагрузки (AJAX)
- Модальное окно с полным текстом новости
- Поиск по тексту новостей в реальном времени

**Использование:**
1. Откройте `http://localhost/ajax-lab/news.html`
2. Новости загрузятся автоматически
3. Используйте пагинацию для навигации между страницами
4. Нажмите "More info" для просмотра полного текста
5. Используйте поле поиска для фильтрации новостей

**Технические особенности:**
- POST запрос к `ajax.php?type=news`
- Пагинация с кнопками Prev, Next и номерами страниц
- Модальное окно с анимацией появления
- Динамическая фильтрация без перезагрузки
- Плавная прокрутка к началу списка при смене страницы

## API Endpoints

### Получение данных nickname

```
URL: ajax.php?type=nickname
Method: POST
Response: JSON массив с nickname и name
```

### Получение новостей

```
URL: ajax.php?type=news
Method: POST
Response: JSON массив с title, text и full_text
```

## Особенности реализации

### AJAX запросы

Используется современный Fetch API вместо XMLHttpRequest:

```javascript
fetch('ajax.php?type=news', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/x-www-form-urlencoded',
    }
})
.then(response => response.json())
.then(data => {
    // Обработка данных
})
.catch(error => {
    // Обработка ошибок
});
```

### Пагинация без перезагрузки

Данные нарезаются на страницы динамически:

```javascript
const startIndex = (currentPage - 1) * newsPerPage;
const endIndex = startIndex + newsPerPage;
const pageNews = filteredNews.slice(startIndex, endIndex);
```

### Поиск в реальном времени

Фильтрация данных при вводе в поле поиска:

```javascript
newsSearch.addEventListener('input', (e) => {
    const query = e.target.value.toLowerCase();
    filteredNews = allNews.filter(news => 
        news.title.includes(query) || 
        news.text.includes(query)
    );
    displayNews();
});
```
