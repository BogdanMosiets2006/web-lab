## Описание решений

### Задание 1: Рекурсивное возведение в степень
**Функция:** `power($val, $pow)`

Реализована рекурсивная функция возведения числа в степень:
- Базовый случай: если степень = 0, возвращает 1
- Для отрицательных степеней: возвращает 1/power($val, -$pow)
- Рекурсивный случай: $val * power($val, $pow - 1)

### Задание 2: Текущее время с правильными склонениями
**Функция:** `getCurrentTime()`

- Получает текущие часы и минуты
- Использует функцию `declension()` для правильного склонения слов "час" и "минута"
- Склонение основано на правилах русского языка (1 час, 2 часа, 5 часов и т.д.)

### Задание 3: Цикл while для чисел, делящихся на 3
Использует цикл `while` для перебора чисел от 0 до 100 и проверяет делимость на 3 через оператор `%` (остаток от деления).

### Задание 4: Цикл do-while для описания чисел
Использует цикл `do-while` для вывода чисел от 0 до 10 с определением:
- 0 - "это ноль"
- Четные числа
- Нечетные числа

### Задание 5: Цикл for без тела
Реализован хитрый способ - все действия вынесены в секцию инкремента цикла:
```php
for($i = 0; $i < 10; echo $i++, ($i < 10 ? ', ' : ''));
```

### Задание 6: Массив областей и городов
Создан многомерный ассоциативный массив:
- Ключи - названия областей
- Значения - массивы с городами
- Вывод через вложенные циклы `foreach`

### Задание 7: Города на букву "К"
Расширение задания 6 с фильтрацией:
- Использует функцию `mb_substr()` для работы с кириллицей
- Проверяет первую букву каждого города

### Задание 8: Транслитерация
**Функция:** `transliterate($string)`

- Создан массив соответствий русских букв латинским
- Используется функция `strtr()` для замены символов
- Поддерживает как строчные, так и заглавные буквы

### Задание 9: Транслитерация с заменой пробелов
**Функция:** `transliterateAndReplace($string)`

- Использует функцию `transliterate()` из задания 8
- Дополнительно заменяет пробелы на подчеркивания через `str_replace()`

### Задание 10: Калькулятор
Полнофункциональный калькулятор:
- HTML-форма с двумя полями ввода
- Кнопки операций: +, -, ×, ÷
- Обработка POST-запроса на PHP
- Сохранение введенных данных после отправки формы
- Защита от деления на ноль
- Валидация ввода (проверка, что введены числа)

### Задание 11: Загрузка файлов
Форма для загрузки файлов:
- HTML-форма с `enctype="multipart/form-data"`
- Обработка загруженного файла через `$_FILES`
- Создание директории `uploads/` если её нет
- Сохранение файла на сервере
- Вывод имени загруженного файла

## Технические детали

### Используемые возможности PHP:
- Рекурсия
- Циклы: while, do-while, for, foreach
- Массивы (индексные и ассоциативные)
- Функции работы со строками: strtr(), str_replace(), mb_substr()
- Работа с датой и временем: date()
- Обработка форм: $_POST, $_FILES
- Работа с файловой системой: move_uploaded_file(), mkdir()
- Условные операторы: if-else, switch-case

### Безопасность:
- Использование `htmlspecialchars()` для предотвращения XSS-атак
- Проверка наличия данных перед обработкой
- Валидация типов данных (is_numeric)
