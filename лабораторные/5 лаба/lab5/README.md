# Лабораторная работа №5 - JavaScript Компоненты

**Тема:** Основы JavaScript и проектирование компонентов  
**Цель:** Ознакомление с синтаксисом и возможностями языка JavaScript. Закрепление навыков верстки и стилизации. Проектирование и реализация независимых компонентов веб-страницы.

## Описание проекта

Проект представляет собой набор из 7 интерактивных веб-компонентов, реализованных на чистом JavaScript без использования сторонних библиотек и фреймворков.

## Реализованные компоненты

1. **Счётчик кликов** - интерактивный счётчик с статистикой и сохранением данных
2. **Калькулятор** - полнофункциональный калькулятор с поддержкой клавиатуры
3. **Слайдер изображений** - галерея с автопрокруткой и навигацией
4. **Переключатель языка** - мультиязычный интерфейс (RU, EN, ES, DE)
5. **Генератор случайных чисел** - генератор с настройками диапазона и историей
6. **Показать/Скрыть пароль** - компонент с индикатором силы пароля
7. **Фильтрация элементов** - фильтрация по категориям с поиском

## Запуск проекта

1. Клонируйте репозиторий.
2. Откройте файл `index.html` в браузере.

## Особенности реализации

### Технологии
- **HTML5** - семантическая разметка
- **CSS3** - неоморфный дизайн, анимации, градиенты
- **Vanilla JavaScript** - без использования библиотек
- **LocalStorage** - сохранение данных пользователя

### Дизайн
- Современный неоморфный стиль
- Адаптивная верстка
- Плавные анимации и переходы
- Кастомные шрифты (Poppins, JetBrains Mono)
- Цветовая схема с акцентами

### Функциональность
- ООП подход (классы ES6)
- Event-driven архитектура
- Валидация данных
- Обработка ошибок
- Поддержка клавиатуры
- Сохранение состояния

## Компоненты в деталях

### 1. Счётчик кликов
- Увеличение/уменьшение значения
- Статистика кликов
- Сохранение в localStorage
- Анимация при изменении

### 2. Калькулятор
- Базовые арифметические операции (+, -, ×, ÷, %)
- История вычислений
- Поддержка клавиатуры
- Отображение промежуточных результатов

### 3. Слайдер изображений
- Автоматическая прокрутка
- Навигация кнопками и клавишами
- Индикаторы слайдов
- Пауза при наведении

### 4. Переключатель языка
- 4 языка (Русский, English, Español, Deutsch)
- Сохранение выбора
- Динамическое обновление контента
- Уведомления о смене языка

### 5. Генератор случайных чисел
- Настройка диапазона
- Генерация нескольких чисел
- История генераций
- Валидация ввода

### 6. Показать/Скрыть пароль
- Переключение видимости
- Индикатор силы пароля
- Проверка требований безопасности
- Примеры надёжных паролей

### 7. Фильтрация элементов
- Фильтрация по категориям
- Поиск по названию
- Динамическая статистика
- Анимация появления элементов

## Требования

- Современный браузер с поддержкой ES6
- Не требует установки зависимостей
- Работает офлайн (после загрузки)

## Использованные возможности JavaScript

- Классы и ООП
- Event listeners
- DOM manipulation
- LocalStorage API
- Template literals
- Arrow functions
- Destructuring
- Array methods (map, filter, forEach)
- Async/await (для будущих расширений)
