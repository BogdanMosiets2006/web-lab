# PWA Приложение "Суши Доставка"

Лабораторная работа №4 по веб-программированию: Адаптивная верстка прогрессивных веб-приложений

## Описание

Progressive Web Application (PWA) для сервиса заказа суши с полным функционалом мобильного приложения.

## Основные возможности

- Полная адаптивность под мобильные устройства
- Кеширование всех ресурсов (изображения, CSS, JS)
- Предложение установки при первом посещении
- Корзина с localStorage
- Счетчик обратного отсчета акции
- Отзывы в формате "Stories"
- БЭМ методология в CSS
- Bootstrap 5.3

## Установка и запуск

### Шаг 1: Установка сервера

**XAMPP:**
1. Скачайте XAMPP с официального сайта
2. Установите и запустите Apache

### Шаг 2: Запуск приложения

1. Перейти в папку проекта.
2. Открыть файл index.html в браузере.

### Шаг 3: Проверка PWA

1. Откройте DevTools (F12)
2. Перейдите в Application → Service Workers
3. Проверьте, что Service Worker активен
4. Application → Manifest - проверьте манифест
5. Application → Cache Storage - проверьте кеш

## БЭМ Методология

Проект использует методологию БЭМ (Блок-Элемент-Модификатор):

```css
/* Блок */
.product-card { }

/* Элемент */
.product-card__title { }
.product-card__price { }

/* Модификатор */
.product-card--featured { }
```

**Примеры блоков в проекте:**
- `header` (шапка сайта)
- `hero` (главный баннер)
- `category-card` (карточка категории)
- `story-card` (карточка отзыва)
- `product-card` (карточка товара)
- `countdown` (таймер)
- `footer` (подвал)

## Service Worker и кеширование

Service Worker (`sw.js`) реализует стратегии кеширования:

1. **Cache First** - для статических ресурсов
2. **Network First** - для HTML страниц
3. **Cache with Network Fallback** - для изображений

**Кешируемые ресурсы:**
- ✅ Все HTML страницы
- ✅ CSS файлы
- ✅ JavaScript файлы
- ✅ Изображения (jpg, png, svg, webp)
- ✅ Bootstrap CSS/JS

## Особенности реализации
