# Иконки для PWA

Разместите здесь иконки приложения в следующих размерах:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Как создать иконки:

1. Создайте основное изображение размером 512x512px
2. Используйте онлайн-сервисы для генерации всех размеров:
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator
   
3. Или используйте инструменты командной строки:
   - ImageMagick: `convert icon.png -resize 192x192 icon-192x192.png`

## Требования к иконкам:

- Формат: PNG
- Фон: прозрачный или цветной (#e74c3c)
- Содержимое: логотип приложения или значок корзины с суши
- Отступы: 10% от размера для безопасной зоны
